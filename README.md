# Add, Update, or Remove Hyperlinks in PDF using C#

The Syncfusion .NET PDF Library provides a comprehensive suite of tools for managing hyperlinks within PDF documents in your .NET applications. This collection of samples demonstrates various techniques for adding, updating, or removing hyperlinks in PDFs, enabling developers to enhance document interactivity and streamline navigation.

 Sample name | Description
------|------
[Adding hyperlinks (external URLs) to the PDF](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/adding-hyperlinks-to-pdf) |  Demonstrates how to add hyperlinks to external websites or resources within a PDF file using C#. These links allow users to easily access related content or information directly from the PDF.
 [Inserting hyperlinks into an existing PDF file](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/insert-hyperlink-into-an-existing-pdf) | Shows how to add hyperlinks to an already existing PDF file, integrating new links without needing to recreate the document. This is useful for updating content or providing additional resources.
[Creating internal navigation links within a PDF](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/internal-navigation-links-in-pdf) | Explains how to create links that navigate to specific sections or pages within the same PDF document, helping users move quickly through long documents.
[Navigating external documents from a PDF](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/navigate-external-document-from-pdf) | Demonstrates how to link to external documents (such as images, Word files, or text files) from within a PDF using C#. This feature is helpful for adding supplementary resources.
[Navigating specific pages in an external PDF file](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/navigate-pages-to-external-pdf) | Illustrates how to create links within a PDF that point to specific pages of another PDF document, providing detailed navigation options across multiple PDFs.
[Modifying existing hyperlinks in a PDF](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/modify-hyperlinks-in-pdf) | Shows how to update existing hyperlinks within a PDF document, allowing for easy link maintenance and content updates.
[Removing hyperlinks from a PDF document](https://github.com/SyncfusionExamples/pdf-add-update-remove-hyperlinks/tree/master/remove-hyperlink-in-pdf) | Demonstrates how to remove unwanted or outdated hyperlinks from a PDF document, offering flexibility in maintaining and revising the document's link structure.

## How to run the examples

* Download this project to a location in your disk.
* Open the solution file using Visual Studio.
* Rebuild the solution to install the required NuGet package.
* Run the application.
